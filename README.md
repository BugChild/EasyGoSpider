# EasyGoSpider
### “宜出行”热力图数据爬虫: 	

-----

> 
1. 效果受制于部署账号数量，需自行到某宝购买  :pill:
2. yundama部分的代码没有上传，如有需要请自行搜索相关api

### 已完成：
* 部署多账号，自动登陆、识别验证码（手动识别或借助yumdama平台）、获取cookies
* 指定区域爬取（已测试全成都市范围30天稳定运行）
* 断点续爬
* 定时周期爬取（当前一个小时内的数据算一次任务，每两小时一次）

### 基于：
* python27
* scrapy
* mongodb
